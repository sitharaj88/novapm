# NovaPM

**Next-generation AI-powered process manager**

NovaPM is a modern, production-grade replacement for PM2. It features AI-driven auto-scaling
and anomaly detection, a beautiful web dashboard, multi-server distributed management,
and a plugin ecosystem.

## Quick Start

```bash
# Install
npm install -g novapm

# Start a process
nova start app.js --name my-app

# View all processes
nova list

# Open monitoring dashboard
nova dashboard --open
```
