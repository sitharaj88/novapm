# NovaPM

**Next-generation AI-powered process manager for Node.js**

NovaPM is a modern, production-grade replacement for PM2. It features AI-driven auto-scaling
and anomaly detection, a beautiful web dashboard with dark/light theme, multi-server distributed
management, and a plugin ecosystem -- all built in TypeScript with strict mode.

## Quick Start

```bash
# Install globally
npm install -g @novapm/cli

# Start a process
nova-pm start app.js --name my-app

# View all processes
nova-pm list

# Open monitoring dashboard
nova-pm dashboard --open
```

## Why NovaPM?

| Feature | PM2 | NovaPM |
|---------|-----|--------|
| Process Management | Yes | Yes |
| Cluster Mode | Yes | Yes |
| Log Management | Yes | Enhanced (JSON, rotation, streaming) |
| Web Dashboard | Basic | Real-time, mobile-responsive, dark/light theme |
| AI Anomaly Detection | No | Yes (Z-Score, IQR, Pattern, Rate-of-Change) |
| Auto-Scaling | No | Yes (predictive + threshold-based) |
| Natural Language Ops | No | Yes (OpenAI, Anthropic, Ollama) |
| Plugin System | No | Yes (6 official plugins) |
| Multi-Server | Limited | Full agent-controller architecture |
| Health Checks | Basic | HTTP, TCP, Script-based |
| TypeScript | Partial | Full strict mode |
| Metrics Storage | None | SQLite time-series with downsampling |

## Key Features

- **Process Management** -- Start, stop, restart, scale, and monitor Node.js applications
- **Web Dashboard** -- Real-time UI with charts, logs, dark/light theme, and mobile support
- **AI Engine** -- Anomaly detection, auto-scaling, performance advisor, and LLM integration
- **Multi-Server** -- Agent-controller architecture with rolling, canary, and blue-green deploys
- **Plugin Ecosystem** -- Slack, Discord, Email, Prometheus, Docker, and GitHub Deploy plugins
- **Health Checks** -- HTTP, TCP, and script-based monitoring with auto-restart
- **REST API & WebSocket** -- Full API for programmatic access and real-time data streaming
- **PM2 Compatible** -- Reads `ecosystem.config.js` for easy migration

## Architecture

NovaPM is a monorepo with 13 packages:

```
packages/
├── shared/       # Types, constants, Zod schemas
├── core/         # Daemon, process manager, HTTP API, dashboard serving
├── cli/          # Command-line interface (16 commands)
├── dashboard/    # Next.js web dashboard (static export)
├── ai-engine/    # Anomaly detection, auto-scaling, LLM
├── agent/        # Multi-server agent-controller
└── plugin-sdk/   # Plugin development SDK

plugins/
├── plugin-slack/           # Slack notifications
├── plugin-discord/         # Discord notifications
├── plugin-email/           # Email notifications
├── plugin-prometheus/      # Prometheus exporter
├── plugin-docker/          # Docker awareness
└── plugin-github-deploy/   # GitHub deployments
```
