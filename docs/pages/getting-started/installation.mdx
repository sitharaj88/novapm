# Installation

## Requirements

- **Node.js** 20 or later
- **npm**, **pnpm**, or **yarn** package manager

## Install from npm

```bash
npm install -g @novapm/cli
```

Or with pnpm:

```bash
pnpm add -g @novapm/cli
```

## Verify Installation

```bash
nova-pm --version
nova-pm doctor
```

## Startup Script

To ensure NovaPM starts on system boot:

```bash
nova-pm startup
```

This generates the appropriate startup script for your OS:
- **macOS**: launchd plist
- **Linux**: systemd service
- **Windows**: Windows Service (coming soon)
